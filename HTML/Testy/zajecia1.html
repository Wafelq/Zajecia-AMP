Tworzenie stron internetowych

HTML ( ang. HyperText Markup Language ) - język znaczników służący do tworzenia struktury i treści strony

CSS(ang. Cascading Style Sheets) - język opisujący wygląd strony np. koloru, czcionki, wymiarów

JavaScript - skryptowy język dodający interakcje na naszej stronie

Adres URL (ang. Uniform Resource Locator ) - zapis adresów stron np. https://www.youtube.com

WWW (ang. World Wide Web ) - ogólno światowa sieć informacyjna oparta na dokumentach hipertekstowych
